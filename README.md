# Tripo Games Website

A fully responsive, dynamic, and interactive game showcase website built with **HTML, CSS, and JavaScript**.  
This project features multiple pages including **Home, Game Details, Careers, Job Application, and About**, with modern UI/UX interactions, animations, and email integration.

## 🚀 Features

### Home Page
- Featured games with clickable cards
- Hero section with scroll animations

### Game Page
- Dynamic game details from `gamesDetails` data
- Screenshots viewer with next/previous navigation
- User reviews section with total rating calculation
- Promo highlights section
- Swiper slider for multiple content sections

### Careers Page
- List of available jobs
- Click to view detailed job information

### Job Page
- Detailed job description (What you'll do / Who you are)
- SEO meta tags updated dynamically
- Apply form integrated with EmailJS

### Forms
- Dynamic load of application forms
- Phone input plugin with international formatting
- Form validation and submission with EmailJS
- Success and error notifications

### Animations
- Scroll-based animations with **AOS**
- Smooth scrolling effects and "scroll to top" button

### Other Utilities
- Modular JS structure with page-specific initialization
- Header and footer components reused across pages

---

## 💻 Technologies Used

- HTML5, CSS3, JavaScript (ES6 Modules)
- EmailJS for form submissions
- AOS (Animate on Scroll) library
- SwiperJS for sliders
- Intl-tel-input plugin for phone input

---

## 📂 Data

- `gamesDetails` → Array of game objects with images, reviews, and promo text
- `careerJobs` → Array of job objects with job details and description
